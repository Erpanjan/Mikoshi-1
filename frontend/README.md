# ARK Product Vision Demo

A high-fidelity mobile product vision demonstration for ARK, built with Next.js and Tailwind CSS.

## Overview

This project showcases a premium mobile experience for a guided financial journey, from initial discovery through biometric account opening, AI-guided consultation, and finally to policy execution and monitoring.

## Key Features

- **ARK Header & Branding**: Minimalist, premium entry experience.
- **Biometric Onboarding**: Simulated high-security account opening flow.
- **AI Consultation**: Interactive voice/chat interface for financial planning.
- **Policy Visualization**: Detailed breakdown of investment strategies and impact.
- **Execution Flow**: Bank-grade secure authorization demonstration.
- **Active Monitoring**: Post-execution AI agent interaction and portfolio tracking.

## Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: React Hooks & Context API
- **Typography**: Inter (Premium Sans-serif)

## Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or pnpm

### Installation

```bash
npm install
# or
pnpm install
```

### Development

```bash
npm run dev
# or
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Next.js App Router pages and global styles.
- `components/`: Core UI components and screen implementations.
- `context/`: Application-wide context providers (e.g., Language).
- `data/`: Translation files and static data models.
- `public/`: Static assets and images.
